Name clips for personalized cheering (see script.js `playCheerAudio`).

Each file is ONE name spoken alone (e.g. "Arsya!"), same voice/tone as the
praise/encourage clips in `../praise/` and `../encourage/`. It plays first,
then chains into a random generic praise/encourage line — so the two
splice into something like: "Arsya! That's exactly right, keep it up!"

If a player's clip is missing, playback falls back to the generic line only
(no crash) — so files can be added incrementally.
